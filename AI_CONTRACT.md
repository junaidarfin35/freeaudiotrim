# FreeAudioTrim AI Development Contract

This file defines the permanent working rules for AI agents
(Codex, ChatGPT, or other assistants) working on this project.

## Core Rule
Do not modify tool functionality.

## Upload Component
Upload box must be identical everywhere.

## Icon System
Icons must inherit color from CSS variables.

## FAQ System
All FAQ sections must use the same accordion UI.

## Reporting
Every AI modification must return a report including:
- issues found
- changes made
- files modified
- code changes
- functionality safety check
