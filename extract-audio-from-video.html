How it works

The browser reads the file as an ArrayBuffer and attempts to decode the audio track via the Web Audio API's decodeAudioData. For common containers like MP4 with AAC audio, modern browsers can decode the audio track directly. Once decoded into an AudioBuffer, it is exported as WAV and offered for download ‚Äî all in memory in the browser.
