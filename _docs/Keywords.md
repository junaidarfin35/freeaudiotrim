## Note on methodology

This file captures keywords that Happy Scribe, TurboScribe, and MP3Cut appear to target and rank for based on:

- live indexed pages
- public search result visibility
- on-page titles, headings, FAQs, support docs, and tool pages

This is a **manual SERP-evidence list**, not a paid-tool export from Ahrefs/Semrush/Similarweb. So treat these as **confirmed target/ranking themes**, not exact ranking positions.

## Observations

- Happy Scribe has broader coverage around subtitles, translation, dubbing, and editorial-style content.
- TurboScribe has tighter coverage around transcription, utility pages, and free-tool search intent.
- MP3Cut is much more editing-focused and less transcription-focused, with strong utility intent around trimming, ringtone creation, speed, pitch, and extraction.
- The biggest shared keyword overlap sits in:
  - transcription
  - subtitle generation
  - YouTube transcript use cases
  - export-format keywords
  - multilingual transcription/translation queries

- The biggest keyword opportunity added by MP3Cut sits in:
  - audio trimming
  - audio cutting
  - ringtone creation
  - pitch changing
  - speed changing
  - audio extraction from video
